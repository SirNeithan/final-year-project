PAYMENT SYSTEM SETUP INSTRUCTIONS
==================================

1. UPDATE DATABASE SCHEMA
   Run this command in your MySQL/MariaDB:
   
   mysql -u root -p smartdine < data/add_transaction_id.sql
   
   Or manually run this SQL:
   ALTER TABLE orders ADD COLUMN transaction_id VARCHAR(100) NULL AFTER payment_method;

2. UPDATE PRODUCT PRICES IN DATABASE
   If you already have products in your database, run:
   
   mysql -u root -p smartdine < data/update_prices.sql
   
   This will update all product prices to the new range (20,000 - 80,000 UGX)
   
   OR if setting up fresh database:
   
   mysql -u root -p < data/smartdine.sql

3. TEST THE PAYMENT SYSTEM
